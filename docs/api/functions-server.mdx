# Server Functions

## compile

**<code>compile(options: [CompileOptions](/api/types/compileoptions)): Promise&lt;void&gt;</code>**

Compiles a GoatDB application into a standalone executable.

This function performs the following steps:
1. Bundles client-side code (JS/TS/TSX) into a single JavaScript file
2. Processes static assets (HTML, CSS, images, etc.)
3. Generates build information
4. Compiles the server entry point into a native executable for the target
   platform

## startDebugServer

**<code>startDebugServer(options: [DebugServerOptions](/api/types/debugserveroptions)&lt;US&gt;): Promise&lt;never&gt;</code>**

Starts a local debug server. The debug server implements a live reload that
automatically transpiles TypeScript and JSX using ESBuild.
