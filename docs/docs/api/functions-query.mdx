# query Functions

## generateQueryId

**<code>generateQueryId(source: [QuerySource](/api/types/querysource)&lt;&gt;, predicate: [Predicate](/api/types/predicate)&lt;IS, CTX&gt;, sortDescriptor: keyof OS[any] | [SortDescriptor](/api/types/sortdescriptor)&lt;OS, CTX&gt;, ctx: CTX, ns: string): string</code>**

Generates a unique identifier for a query based on its configuration.
The ID is deterministic and will be the same for queries with identical:
- Source (repository or path)
- Predicate function
- Sort descriptor
- Context data
- Schema namespace
