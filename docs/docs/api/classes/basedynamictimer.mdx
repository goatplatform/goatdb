# BaseDynamicTimer

Base timer for timers that dynamically change their fire interval between
a specified range.

**Extends:** [BaseTimer](/api/classes/basetimer)

**Subclasses:** [EaseInExpoTimer](/api/classes/easeinexpotimer), [EaseInOutSineTimer](/api/classes/easeinoutsinetimer)

## Constructor

**<code>new BaseDynamicTimer(minFreqMs: number, maxFreqMs: number, durationMs: number, callback: [TimerCallback](/api/interfaces/timercallback), repeat: boolean, name: string, startAtMax: boolean)</code>**

## Methods

### calcNextFireDate()

**<code>calcNextFireDate(): number</code>**

Calculates the next timestamp in milliseconds when this timer should fire.
This is called internally by schedule() to determine when to fire the
timer.

### reset()

**<code>reset(): [Timer](/api/interfaces/timer)</code>**

Resets the timer to fire at the next appropriate time.
If the timer is not currently scheduled, this is a no-op.

### run()

**<code>run(): boolean</code>**

Executes the timer's callback and handles rescheduling if necessary.

### schedule()

**<code>schedule(): [BaseDynamicTimer](/api/classes/basedynamictimer)&lt;&gt;</code>**

Schedules this timer to fire at the next appropriate time.
If the timer is already scheduled, this is a no-op.

### timingFunc()

**<code>timingFunc(f: number): number</code>**

### unschedule()

**<code>unschedule(): [BaseDynamicTimer](/api/classes/basedynamictimer)&lt;&gt;</code>**

Unschedules this timer if it is currently scheduled.
If the timer is not scheduled, this is a no-op.
