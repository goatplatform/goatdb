# ConsoleLogStream

A LogStream implementation that writes log entries to the console.
Different severity levels are mapped to different console methods.

**Implements:** [LogStream](/docs/api/interfaces/logstream)

## Constructor

**<code>new ConsoleLogStream(severity: number | Severity)</code>**

Creates a new ConsoleLogStream

## Methods

### appendEntry()

**<code>appendEntry(e: NormalizedLogEntry&lt;LogEntry&gt;): void</code>**

Appends a log entry to the console
