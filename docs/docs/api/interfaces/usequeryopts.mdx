# UseQueryOpts

Options for the useQuery hook.

**Extends:** [Omit](/api/interfaces/omit)

## Properties

### showIntermittentResults

```typescript
showIntermittentResults: boolean
```

If `true`, updates UI during the initial scan. If `false`, waits until
scanning is complete.
