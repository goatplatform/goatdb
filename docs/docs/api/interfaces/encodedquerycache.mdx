# EncodedQueryCache

Represents a serialized query cache that can be stored and loaded.

**Extends:** [JSONObject](/api/interfaces/jsonobject)

## Properties

### age

```typescript
age: number
```

The age of the cache when it was created

### results

```typescript
results: string[]
```

Array of item keys that matched the query criteria
