# QuerySource

The source of a query, which can be a repository, another query, or a string
representing a repository path.

## Definition

**Type:** <code>QuerySource = [Repository](/api/classes/repository) | [Query](/api/classes/query)&lt;IS, OS, [ReadonlyJSONValue](/api/types/readonlyjsonvalue)&gt; | string</code>
